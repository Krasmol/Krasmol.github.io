## 📋 О проекте

Это мой персональный сайт-портфолио, созданный для демонстрации моих навыков и проектов. Сайт выполнен в современном **dark core** стиле с градиентными акцентами и адаптивным дизайном.

- 🎨 **Современный дизайн** в тёмной теме
